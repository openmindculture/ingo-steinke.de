React Web App with React Component Library

  • Role: Senior Frontend Developer (React)
  • Customer: wao.io (Sevenval/Avenga)
  • Industry: SaaS / Internet-Dienstleistungen (B2B)
  • Time: 07/2017 – 10/2020
  • Project: React web app with React component library for UI configuration and data visualisation of website optimisations
  • Tech Stack: React · JSX · JavaScript · ES6 · Storybook · CSS in JS · Git · GitLab
  • Challenges:
    • Development and maintenance of modular UI components in React
    • Visualisation of data and configurations in real-time (React state management)
    • Develoment and maintenance of Storybook documentation und modular test automation

The SaaS platform wao.io was used to configure various optimization options regarding loading speed (web performance) and web security for existing websites and to visualize the expected and achieved benefits graphically. Thanks to a modular React architecture, we were able to expand an initially rudimentary UI into a complex interface with detailed documentation and numerous landing pages. To facilitate integratinng existing projects for our customers, we documented use cases for well-known hosts step by step and provided plugins for well-known software such as WordPress and Shopware. It was my responsibility to develop and publish the wao.io Cache Control Plugin for Shopware. wao.io was used in over 20 customer projects by Sevenval (now Avenga) and is currently no longer available to new customers.

  • Higher willingness to buy thanks to faster loading spped
  • Conversion Rate increased by 12% (according to customer feedback)
  • Integration with popular CMS and e-commerce software
